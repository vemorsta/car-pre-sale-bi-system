# Dealers Dashboard

## Purpose
Analyzes pre-sale performance across dealerships to compare activity and financial
contribution.

## Key Analyses
- Total Down Payment by Dealer
- Number of Purchase Requests per Dealer
- Dealer performance comparison across regions

## Visual Components
- Bar Charts for dealer-level comparison
- Table for Dealer performance comparison across regions by Avg Down Payment Per Requests

## Filters & Slicers
- Car Model
- Category
- Date

## Interactivity
- Drill-through enabled to the DealerDetails dashboard

## Business Value
- Identifying high- and low-performing dealers
- Supporting dealer evaluation and incentive planning
- Enabling regional sales performance analysis
