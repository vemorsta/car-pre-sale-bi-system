# DealerDetails Dashboard

## Purpose
Provides a detailed drill-through analysis for an individual dealer selected
from the Dealers dashboard.

## Key KPIs
- Dealer Total Down Payment
- Dealer Total Purchase Requests
- Average Down Payment per Request

## Visual Components
- KPI Cards for dealer-specific metrics
- Table listing customers associated with the selected dealer
- Line Chart showing dealer Down Payment trend over time
- Donut Chart displaying car model share for the dealer

## Interactivity
- Accessed exclusively via Drill-through from Dealers dashboard
- Fully context-aware based on selected dealer

## Business Value
- Deep performance analysis at the dealer level
- Understanding dealer-specific sales patterns
- Supporting targeted dealer-level decisions
